## 🔍 Análise de Profundidade

### Por Nível de Aprendizado

**INICIANTE (0-3 meses)**:
- Leia: README + MINIGUIA (Nível Básico)
- Use: PROMPTS 1.1 (análise FII) + 3.1 (devo comprar)
- Refira: GLOSSARIO conforme necessário
- Tempo: 5-10 horas

**INTERMEDIÁRIO (3-12 meses)**:
- Leia: MINIGUIA (Básico + Intermediário)
- Use: PROMPTS (categorias 2-5)
- Estude: PROMPTS_E_CICATRIZES para melhorar análise
- Tempo: 20-30 horas

**AVANÇADO (1+ anos)**:
- Leia: MINIGUIA (todos os níveis)
- Use: PROMPTS (todas as categorias)
- Domine: PROMPTS_E_CICATRIZES para criar novos prompts
- Customize: Glossário conforme especialização
- Tempo: 40+ horas
